Protect your vision

'Protect your vision' is project built with Javascript, HTML and CSS. It is count down application. Allows user to select time - 20, 40, 60 minutes. Once time is selected, application starts count time down and displays remaining time. Music starts to play once the time has passed. Gentle sound after selected time is reminder that eyes need break and remind user to relax or alternatively do easy eye exercises.
