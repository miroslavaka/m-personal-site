Protect your vision

'Protect your vision' is project built with Javascript, HTML and CSS. It is count down application. Allows user to select time - 20, 40, 60 minutes. Once time is selected, application starts count time down and displays remaining time. Gentle sound reminds you to take break and relax for a while to renew your energy for next time slot.

Live Demo See code on Github
